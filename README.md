## 📌 About

This repository contains my solutions to Codeforces problems, organized by problem ID and difficulty/topic. Each solution includes:

- The approach and reasoning behind it
- Time & space complexity analysis
- Clean, commented code

I use this repo to track my progress, revisit patterns I've learned, and build a reference I (and others) can come back to.

## 🗂️ Structure

```
solutions/
├── <problem-id>/
│   ├── solution.cpp (or .py)
│   └── notes.md          # approach, complexity, edge cases
```

Example: `solutions/4A/` contains my solution to [Codeforces 4A - Watermelon](https://codeforces.com/problemset/problem/4/A).

## 🚀 How to Use

Each solution folder is self-contained. To run a C++ solution:

```bash
g++ -std=c++17 -O2 -o solution solution.cpp
./solution < input.txt
```

To run a Python solution:

```bash
python3 solution.py < input.txt
```
